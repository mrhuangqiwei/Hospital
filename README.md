#### Hosiptal 医院
* 后端代码：/src/
* 前端代码：/frontend/
* web目录：/WebRoot

#### 配置
nodejs + tomcat/jetty

#### 前端：
* 编译
```
cd frontend
npm install
npm run build
```
代码生成到/WebRoot/下，以方便java的服务器访问

* 调试
```
cd frontend
npm run dev
```

参考：https://github.com/mrhuangqiwei/Hospital/blob/master/frontend/README.md
